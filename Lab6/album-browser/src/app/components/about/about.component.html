About Album Browser

Application Info

Album Browser is a single-page application built with Angular that allows users to browse, view, and manage photo albums. Data is fetched from the JSONPlaceholder API.

  • Framework: Angular 17+ (Standalone Components)
  • API: JSONPlaceholder REST API
  • Features: Routing, HTTP Client, CRUD operations

Course Info

  • Course: Web Development / Angular Lab 6
  • Assignment: Angular Routing & HTTP Client

Tech Stack

  • Angular Router for client-side navigation
  • HttpClient for RESTful API calls
  • RxJS Observables for async data streams
  • TypeScript interfaces for type safety
